# Watch Later Easier (Firefox Extension)

**Watch Later Easier** is a lightweight Firefox extension that simplifies how you save and access YouTube videos. It provides a faster, more direct alternative to manually managing YouTube’s built-in “Watch Later” playlist.

## What It Does

This extension lets you quickly add videos to your *Watch Later* list without navigating through multiple menus on YouTube. It streamlines the workflow so you can save content with minimal friction and get back to browsing.

## Key Features

- One-click saving – Add videos to your Watch Later list instantly  
- Faster workflow – Avoid YouTube’s multi-step UI  
- Lightweight and focused – No unnecessary features or overhead  
- Seamless integration – Works directly within the YouTube interface  

## Why Use It?

YouTube’s default “Watch Later” functionality is useful, but not always efficient. This extension is designed for users who:

- Frequently save videos for later viewing  
- Want a quicker, distraction-free way to bookmark content  
- Prefer minimal UI interactions  

## Installation

Install the extension directly from the Firefox Add-ons store:

https://addons.mozilla.org/en-US/firefox/addon/watch-later-easier/

## How It Works

Once installed:

1. Navigate to any YouTube video  
2. Use the extension’s quick action to add it to your Watch Later list  
3. Access your saved videos as usual via YouTube  

## Notes

- This extension enhances YouTube’s existing Watch Later feature—it does not replace it  
- Requires permissions to interact with YouTube pages
